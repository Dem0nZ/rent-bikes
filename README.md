# VN Bikes Monorepo

## Apps
- `apps/web`: TanStack Start application
- `apps/supplier`: TanStack Start application
- `apps/api`: NestJS API

## Packages
- `packages/ui`
- `packages/db`
- `packages/types`
- `packages/config`
- `packages/email`

## Scripts
- `pnpm dev`: Run all apps in development mode
- `pnpm build`: Build all apps
- `pnpm check-types`: Check types for all packages and apps
